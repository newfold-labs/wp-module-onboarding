# Overview

## What the module does

**wp-module-onboarding** provides the next-generation onboarding experience for WordPress sites on Newfold Digital brands (e.g. Bluehost, HostGator). It guides users through setup steps, site type selection, and integrations (e.g. Facebook, migration). It uses wp-module-onboarding-data, wp-module-patterns, wp-module-facebook, wp-module-migration, Mustache templates, and wp-config-transformer.

## Who maintains it

- **Newfold Labs** (Newfold Digital). Distributed via Newfold Satis.
